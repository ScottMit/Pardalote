Why Pardalote

Made for people who think with their hands

Pardalote was built for design studios and classrooms — places where an idea moves back and forth between a breadboard and a browser tab.

Two languages, equal partners

You write real Arduino code and real JavaScript. Neither side is privileged — both get control of the hardware, and both can lead. Perfect when you're constantly shifting between a physical device and the screen.

Multi-user by default

One Arduino can talk to many browsers, and one browser can talk to many Arduinos. A whole class can connect to the same device — or one interface can conduct a room full of devices.

Always in sync

Connect at any moment and you're up to date: the Arduino tells every new browser its current state the instant it joins. Reload the page mid-project and everything just keeps going.

WiFi, not wires

The Arduino runs the connection, so any device on your network can join — laptop, tablet, phone. No USB cables, no Node.js — WebSockets make connecting and staying connected easy.

Open source

Pardalote is built on free and open source platforms, and it's also free. Published under an open source GPL-3.0 licence - use it, read it, fork it, — it belongs to the people that use it.

How small is small?

A complete Pardalote project

sketch.ino — the whole Arduino side
#include <Pardalote.h>

void setup() {
  Pardalote.begin();
}

void loop() {
  Pardalote.run();
}
sketch.js — blink from the browser
const arduino = new Arduino();
arduino.connect('192.168.1.42');

arduino.on('ready', () => {
  arduino.pinMode(13, OUTPUT);
  arduino.digitalWrite(13, HIGH);
});

The JavaScript API mirrors Arduino's own function names — pinMode, digitalWrite, analogRead — so what you learn on one side carries straight over to the other.

In the classroom

Built for teaching, tested in studios

Pardalote grew out of design education: students prototyping IoT ideas need to see cause and effect quickly, on both sides of the wireless link. It plays especially well with p5.js, building on Processing and Wiring's (Arduino) shared heritage.
