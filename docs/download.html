Setup

Three steps to blinking

Install the Arduino library

Copy the Pardalote folder into your Arduino libraries folder and restart the IDE. It appears under File → Examples → Pardalote. Full details in Installation.

Upload an example and join your WiFi

Open basic-LED, upload it, and give the board your WiFi details — Pardalote asks for them in the Serial Monitor and remembers them. The board then shows you its IP address. See WiFi configuration for all the options.

Open a web page

Put the board's IP address into the example's sketch.js and open index.html in your browser. That's the whole toolchain.

What you need

Boards and requirements

Pardalote runs on the Arduino UNO R4 WiFi and ESP32 development boards. You'll need the Arduino IDE, a web browser, and a text editor — the Arduino and your browser just have to share the same WiFi network.

Some extensions rely on standard Arduino libraries (servo, stepper, NeoPixel support and so on) — the installation guide lists exactly what to install for the hardware you're using.
